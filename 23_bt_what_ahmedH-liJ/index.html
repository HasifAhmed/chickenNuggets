Containers

Containers are extremely usefull in bootstrap. They create a clean and ordered display on sites. Everything here except for the navigation bar is in a container. If you notice, containers even out the amount of content you have per line without you having to do any other work.
Below is a container we created and below that is code for it:

This
Goes
Left
To
Right
<div class="container"> <div class="row"> <div class="col"> This </div> <div class="col"> Goes </div> <div class="col"> Left </div> <div class="col"> To </div> <div class="col"> Right </div> </div> </div>

The maximum number of columns on one row is 12. Any more than that will cause the columns to spill into the next line!

Buttons

Buttons! Buttons! BUTTOOOOOOONS! Bootstrap makes creating nice aesthetic buttons very easy. If you look at the code below, all you have to do is replace the 0000 with the names on the buttons. Each name on the buttons mean a specific button type.

<button type="button" class="btn btn-0000"></button>

There are other types of buttons too! Like disabled and outline buttons

<button type="button" class="btn btn-primary" disabled>Disabled</button> <button type="button" class="btn btn-outline-primary">Outline</button>
